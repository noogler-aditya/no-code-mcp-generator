## â“ Troubleshooting

### "Cannot find module" Error

Make sure you're using the **absolute path** in your Claude Desktop config, not a relative path.

**Wrong:** `"args": ["dist/index.js", "--stdio"]`  
**Right:** `"args": ["/Users/you/my-server/dist/index.js", "--stdio"]`

### "Server disconnected" Error

1. Make sure you ran `npm run build` first
2. Check the path in your config is correct
3. Run `node dist/index.js --stdio` directly to see error messages

### Tools Don't Work / API Errors

The generated server calls the API defined in your OpenAPI spec. Make sure:
1. The API server is running and accessible
2. API keys are set in your `.env` file if required
